This is a simple Task Manager application built using JavaScript and Node.js.
The app runs in the terminal (command line) and helps users manage tasks using basic options like add, view, update, and delete.

This project is made for beginners to understand how JavaScript logic works in real projects, without using HTML, CSS, or any frameworks.
